# Игра "Угадай число"

Это простая игра, в которой программа генерирует случайное число от 1 до 100, и пользователю нужно угадать его. Программа будет подсказывать, больше или меньше число, чем загаданное, до тех пор, пока пользователь не угадает.

## Как запустить
Для того чтобы сыграть в игру, нужно:
1. Склонировать репозиторий.
2. Запустить файл `main.py`:
   ```bash
   python main.py
